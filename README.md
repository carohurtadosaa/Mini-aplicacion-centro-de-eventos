# 🎉 Mini-aplicación Centro de Eventos

## 📌 Descripción
Aplicación construida en **Vue.js** con arquitectura modular, que simula la gestión de un centro de eventos.  
Incluye módulos de login, dashboard de usuario y administración de eventos, con datos simulados en Vuex para cumplir requisitos académicos sin backend.

---

## 🚀 Objetivos del proyecto
- Practicar **binding de formularios** y **manejo de eventos** en Vue.
- Aplicar **convenciones de nombres multi-palabra** en componentes.
- Documentar el proceso con capturas, tablas comparativas y reflexiones técnicas.
- Publicar el proyecto como parte de mi portafolio profesional.

---
